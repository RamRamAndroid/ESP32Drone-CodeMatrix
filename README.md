ESP32 Drone by CodeMatrix is a compact, WiFi-controlled drone based on the ESP32-S3 microcontroller. Designed for hobbyists, makers, and engineers, ESP32 Drone by CodeMatrix offers a simple yet powerful platform for drone experimentation and development. It is an open-hardware project, making it easy to modify and expand.
